# indexpalavras

Trabalho Programação:  

Utilizando os códigos e técnicas vistas em sala de aula indexar os livros em memoria.  

Separando cada palavra e armazenando as linhas de ecorrencia de cada palavra nos livros.  

Depois do processo de indexação e após apresentar os tempos nescessários para cada etapa deve ser liberado um console para consulta.  

Para cada palavra consultada, devem ser exibidas todas as linhas onde a palavra aparece sendo a palavra destacada.  

Alem disso deve ser apresentado o tempo nexessário para realização da busca.
